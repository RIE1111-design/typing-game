# タイピングゲーム

落ちてくる単語を素早く打ち込むタイピングゲームです。

## 遊び方

1. `index.html` をブラウザで開く
2. **スタート** ボタンを押す（または `Enter` キー）
3. 落ちてくる単語を入力して `Enter` で送信
4. 単語が画面下まで落ちると残機が1つ減る
5. 残機が0になるとゲームオーバー

## 単語の種類

| 色 | 種類 | 得点 |
|----|------|------|
| 青 | ノーマル | +1点 |
| 黄 | 速い | +2点 |
| 赤 | 超速い | +3点 |

コンボを繋ぐとボーナス点が加算されます。

## レベルシステム

15点ごとにレベルが上がり、単語の速度と難しさが増します。

## ファイル構成

```
typing-game/
└── index.html   # ゲーム本体（HTML + CSS + JavaScript）
```
